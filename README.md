# AI-Assisted-Patient-Early-Warning-System

## Abstract

The AI-Assisted Patient Early Warning System is a web-based healthcare application developed using Python, Flask, SQLite, and Machine Learning. The system predicts a patient's health condition based on medical parameters such as Age, Gender, Blood Pressure, Cholesterol, Maximum Heart Rate, and Old Peak. Based on the prediction, the patient is classified into Normal, Warning, or Critical categories. The application securely stores patient records in an SQLite database, allowing users to view patient history and generate reports. This system helps healthcare professionals identify high-risk patients at an early stage and supports timely medical decision-making.

## Introduction

Early detection of critical health conditions is important for improving patient care and reducing medical emergencies. Manual monitoring of patients can be time-consuming and may delay treatment.

The AI-Assisted Patient Early Warning System uses Machine Learning to analyze patient health parameters and predict the patient's risk level. The application provides a simple and user-friendly interface for healthcare professionals to enter patient details, obtain predictions, and manage patient records efficiently.

The system includes the following features:

1.User Registration and Login
2.Patient Information Entry
3.AI-Based Risk Prediction
4.Risk Classification
5.SQLite Database Storage
6.Patient History
7.Patient Report Generation

## Methodology

The proposed system follows these steps:

Step 1: User Registration : New users register using a username and password.

Step 2: User Login : Registered users log in using their credentials.

Step 3: Dashboard : After successful login, the dashboard is displayed.

Step 4: Enter Patient Details
The user enters: 
1.Patient Name
2.Age
3.Gender
4.Blood Pressure
5.Cholesterol
6.Maximum Heart Rate
7.Old Peak

Step 5: Data Validation : The system validates and processes the entered patient data.

Step 6: AI Prediction : The Machine Learning model analyzes the patient data and predicts the patient's health condition.

Step 7: Risk Assessment  
The system classifies the patient into one of the following categories:
a) Normal
b) Warning
c) Critical

Step 8: Store Patient Record : The patient's information and prediction result are stored in the SQLite database.

Step 9: Generate Patient Report : A report containing the patient's details and prediction result is generated.

Step 10: View Patient History : Users can view and search previously stored patient records.

Step 11: Delete Record (Optional) : Patient records can be deleted whenever required.
